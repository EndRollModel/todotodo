## 兔嘟莉絲特

因朋友要求加上自己想寫看看 Electron      
所製作的簡易待辦清單程式        
由於自己對網頁與相關架構不熟 開發皆依據Electron手冊 與 Boostrap5操作手冊製作        

### 功能
目前提供以下主要功能    
- 待辦     
  - 完成後勾選可顯示當下完成時間    
- 剪貼簿       
  - 可用於快速的複製文字    
- 群組    
  - 可以將待辦是物語剪貼簿製作成群組
