# 兔嘟莉絲特 (TuduTudu)

<hr>

簡易的待辦事務清單程式 ( 使用electron製作 )


### 功能
目前提供以下主要功能    
- 群組
  - 可以將待辦事務與剪貼簿製作成群組
- 待辦     
  - 完成後勾選可顯示當下完成時間    
- 剪貼簿       
  - 可用於快速的複製文字
